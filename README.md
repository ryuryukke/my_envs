# QuickStart
## jupyterlab
```
$ docker run -it -p 8888:8888 ryuryukke/jupyterlab:latest
```
## jupyterlab-nlp
```
$ docker run -it -p 8888:8888 ryuryukke/jupyterlab-nlp-ja:latest
```
Then, pls access to
```
localhost:8888
```
